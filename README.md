#### ※오목1 예제 실험
1. 최초 실행시 오류 : import numpy 에러 (현재 numpy package가 설치되어 있지 않기 때문에 에러 발생)
> 1번 해결 방법
> 터미널에서 numpy Package 존재여부 체크
> 'pip show numpy'
> numpy Package 설치
> 'pip install numpy'
> 추후 pandas, metplotlib 등 필요한 Package를
> 'pip install 패키지명' 명령을 이용하여 설치해 사용하면 됨
